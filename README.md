# elite-motors
🚗 Elite Motors – Premium Car E-commerce Website

📌 Overview

Elite Motors is a full-stack e-commerce web application for a premium car dealership.
The platform allows users to explore luxury vehicles, manage favorites, add items to a cart, and complete a checkout process.

«🎓 Final Project – Web Programming 2
💻 IT Department | 2025–2026»

---

✨ Features

- 🏠 Home Page (Hero + Featured Cars)
- 🚗 Products Page (Search, Filter, Sort)
- ❤️ Favorites System (Persistent)
- 🛒 Shopping Cart (Dynamic total & quantity)
- 💳 Checkout with validation
- 📱 Fully Responsive Design

---

🛠️ Technologies Used

- React 18 + TypeScript
- Tailwind CSS
- Zustand
- React Router
- Vite
- LocalStorage

---

🎨 Design

Inspired by luxury brands like Porsche & Bentley
Dark Theme + Gold Accent ✨
